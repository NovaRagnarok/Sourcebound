## Task Brief

Decide and document the near-term auth boundary before any auth code lands.
This chunk should state who Sourcebound is for today, what kind of authenticated
usage the next auth work is allowed to target, and which surfaces must become
protected first, while staying explicit that broader public multi-user or
self-serve product modes are still unsupported.
