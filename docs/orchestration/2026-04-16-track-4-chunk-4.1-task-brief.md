# Track 4 Chunk 4.1 Task Brief

## Chunk Identity

- Track id: `Track 4`
- Chunk id: `Chunk 4.1`
- Chunk title: `Supported Single-Host Deployment`
- Backlog id: `SB-015`

## Task Brief

Turn the trusted-operator deployment story into one concrete, supported
single-host path. This chunk should define a real app container with the
in-process worker enabled, plus Postgres, Qdrant, and a reverse proxy,
document it clearly, and make the repo’s health and verification guidance
match that path without implying a broader production hosting promise.
